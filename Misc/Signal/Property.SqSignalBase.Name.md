# Description

The name given to a `SqSignalBase` insance.

# Specifications

* **Aliases**:
```D
SqSignalBase.Name
```

* **Input**:
	* `N/A` *This property is read only.*
* **Output**:
	* `string` *The name given to the signal instance.*
* **Throws**:
	* This property does not throw errors.

# Remarks

Signals can only be named when they're created.
