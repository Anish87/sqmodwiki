# Description

Check whether any slots were bound to a `SqSignalBase` insance.

# Specifications

* **Aliases**:
```D
SqSignalBase.Empty
```

* **Input**:
	* `N/A` *This property is read only.*
* **Output**:
	* `bool` *The value `true` if at least one slot is connected, `false` otherwise.*
* **Throws**:
	* This property does not throw errors.
